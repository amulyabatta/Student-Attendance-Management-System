# Student Attendance Management System

A web-based Student Attendance Management System built using Python, Flask,
SQLite, and HTML.

## Features
- Add student details
- Mark attendance (Present / Absent)
- View attendance records
- SQL database for persistent storage

## Technologies Used
- Python (Flask)
- SQLite (SQL)
- HTML

## How to Run
1. Install Flask:
   pip install flask
2. Run the application:
   python app.py
3. Open browser:
   http://127.0.0.1:5000/
